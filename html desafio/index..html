Tags de Título

As tags de título são utilizadas para definir a hierarquia de títulos em uma página HTML. Elas são usadas para estruturar o conteúdo da página e também para fornecer informações importantes aos mecanismos de busca.

<h1> até <h6>

O HTML fornece seis níveis de títulos, de <h1> (o mais importante) até <h6> (o menos importante). É importante usar os títulos corretamente para garantir a acessibilidade e a semântica correta da página.

Tags de Texto

As tags de texto são usadas para formatar e organizar o conteúdo textual em uma página HTML. Elas permitem adicionar ênfase, estilo e estrutura ao texto.

<p>, <mark>, <small>, <i>, <u>, <strong>

Essas tags são algumas das mais comuns para formatação de texto:

  • <p>: Define um parágrafo de texto.
  • <mark>: Destaca o texto.
  • <small>: Define um texto com tamanho menor.
  • <i>: Coloca o texto em itálico.
  • <u>: Sublinha o texto.
  • <strong>: Define o texto como importante.

Listas

As listas são usadas para organizar itens de forma ordenada ou não ordenada.

<ol>, <ul>, <li>

Existem dois tipos principais de listas:

  • <ol>: Define uma lista ordenada, onde cada item é numerado.
  • <ul>: Define uma lista não ordenada, onde cada item é marcado com um ponto ou outro marcador.
  • <li>: Define um item de lista dentro de uma lista ordenada ou não ordenada.

Links

As tags de link são usadas para criar links para outras páginas ou recursos.

<a>

A tag <a> é usada para definir um hyperlink, que permite ao usuário navegar para outra página da web clicando no link.

Outras Tags

Existem várias outras tags HTML úteis para formatação e estruturação de conteúdo.

<hr>, <sub>, <sup>, <blockquote>, <font>, <del>, <abbr>

Essas tags incluem:

  • <hr>: Cria uma linha horizontal para separar conteúdo.
  • <sub>: Define texto como subscrito.
  • <sup>: Define texto como sobrescrito.
  • <blockquote>: Define uma citação em bloco.
  • <font>: Define a fonte do texto (obsoleta em HTML5).
  • <del>: Define texto deletado.
  • <abbr>: Define uma abreviação ou acrônimo.
